重要说明

此管理面板专为 Cloudflare Pages 静态托管环境设计。由于 Cloudflare Pages 是纯静态托管平台，不支持后端服务器功能，因此部分高级功能（如服务控制、日志查看、系统诊断等）在此环境中不可用。

要使用完整的管理功能，请在支持 Node.js 的服务器环境中部署完整应用程序。

功能限制

由于 Cloudflare Pages 是静态托管平台，以下功能在此环境中不可用：

  • 后端 API 接口调用
  • 服务启动/停止控制
  • 实时日志查看
  • 系统诊断工具
  • 智能部署修复
  • 数据库操作
  • 服务器进程管理

要使用上述功能，请部署完整版应用程序到支持 Node.js 的服务器环境。
